This project is aimed to track the device, like telephone, ipad, computer, etc.

It uses map like google map to show registered coordinates of the device, and also to geocode them.
It also uses geofencing to give alerts if the device is not in the range it is supposed to not go far beyond.

These are some of technologies that are in this project:
========================================================
- Project uses One Time Password (OTP) during registration.
- It uses Bcrypt to hash password.
- It uses Spring security to give access to APIs.
- It uses technologies like Geofencing and Geocoding.
